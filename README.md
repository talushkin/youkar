# YouKar - יוקאר קריוקי

עמוד בקשות ותשלום של יוקאר - קבוצת WA קריוקי

## Tech Stack
- Next.js 14 (TypeScript)
- Tailwind CSS
- RTL Hebrew UI

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features
- 🎤 Song request form (singer, song, requester, notes)
- 📋 Live song queue with status indicators
- 💳 Payment section with total calculation
- 💬 WhatsApp group link
- 🌐 Full Hebrew RTL layout
- 🎨 YouKar brand colors (navy, red, white)
